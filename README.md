# Lightspeed Discord Nuke Bot
Multi-token asynchronous Discord server destruction tool with 35+ commands. Achieves maximum speed by concurrent API calls and rate-limit bypassing.

## Setup
1. Install Python 3.10+
2. `pip install -r requirements.txt`
3. Add bot tokens to `tokens.txt` (one per line)
4. Run `python run.py`
